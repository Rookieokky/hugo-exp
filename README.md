## Quickstart
Portfolio Web Application built with [MERN](http://mern.io/)

Temporarily hosted on Heroku [here](https://hugo-exp.herokuapp.com) 

```
  git clone https://github.com/HugoCe17/hugo-exp.git
  cd hugo-exp
  yarn install
  mongod
  npm start:watch
```
